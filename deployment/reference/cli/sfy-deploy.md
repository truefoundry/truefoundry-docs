# sfy deploy

**Usage**

`sfy deploy [OPTIONS] COMMAND [ARGS]...`

**Summary**

Deploy servicefoundry Service

**Options**

| **Option** | **Description** |
| :--- | :--- |
| -f, --file | path to servicefoundry file |
| -w, --workspace-fqn | workspace to deploy to |
| -h, --help | Show this message and exit. |
